Frequently Asked Questions

Q) How do I upgrade to the New RealArcade?
A) Just click the "Upgrade Now" button above to start your upgrade.

Q) Does the upgrade cost anything?
A) No. The upgrade is absolutely free.

Q) What will happen when I upgrade?
A) Your current RealArcade player will be removed from your computer, and your games will be upgraded to work with the New RealArcade.

Q) Will I be able to use my RealArcade player after I upgrade?
A) No, the RealArcade player will not be available for you to use anymore. Your games will be upgraded to work with the New RealArcade web site, and you will need to use your favorite web browser to find and download games at www.realarcade.com. You can still use the RealArcade desktop icon to get to the New RealArcade, even when offline.

Q) Will the New RealArcade work with Windows Vista?
A) Yes. Windows Vista is fully supported on the New RealArcade.

Q) What are the minimum system requirements for the New RealArcade?
A) You must have either Windows XP or Windows Vista installed as the operating system. Internet Explorer 6 and 7, and Firefox 2 web browsers are supported as well. The minimum requirements for each game are noted on the game page for that game.
